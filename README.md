# EnokBlog Mac桌面仿真博客系统

#### 项目介绍
EnokBlog 博客系统是一款仿真Mac电脑桌面的多窗口博客系统。
#### 技术选型
* 前后端均使用 TypeScript
* 服务端使用 Node.js + Egg.js 构建
* 前端使用 Webpack + React.js 构建
* 数据库使用 MySQL
* 缓存使用 Redis
